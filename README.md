# library-information-system-v3

3rd generation of LibraryInformationSystem sample application

## Building

```
npm install

npm run build --ws
```

## Starting

```
npm run prod --ws
```

## Trying API

http://localhost:5000/docs/#/

## Predefined Users

- Admin: q/q
- Editors: e/e, librarian/librarian
- Borrowers: r/r, borrower/borrower
